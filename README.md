# Sift Sentinel 🚨  
A multi-module project for detecting **cyberbullying** and **deepfakes**.

---

## 📌 Modules
1. **Cyberbullying Detection** (DistilBERT + SHAP explainability)
   - Trains on labeled text data
   - Provides inference with explainability (SHAP waterfall plots)

2. **Deepfake Detection** (coming soon)
